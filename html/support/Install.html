Server Preparation

This section describes the setup on the server that runs the Java programs and makes them available to a browser client.

LAMP Server

Install an Apache server:

sudo apt-get install tasksel

Tomcat

Install Tomcat:

sudo apt-get install tomcat7


Also install

sudo apt-get install tomcat7-docs tomcat7-admin tomcat7-examples

To test the installation:
go to

http://localhost:8080

To use the manager, set up a user.
if you get error " * no JDK or JRE found - please set JAVA_HOME":

nano /etc/default/tomcat7

add the line

JAVA_HOME=/usr/lib/jvm/java-8-oracle

To allow the web management interface:

sudo nano /etc/tomcat7/tomcat-users.xml
and add the lines
<tomcat-users>
<user username="admin" password="password" roles="manager-gui,admin-gui" />
</tomcat-users>

Then restart Tomcat

service tomcat7 restart

If a page takes very long to load, open

nano /etc/default/tomcat7

and add the following line:

JAVA_OPTS="-Djava.security.egd=file:/dev/./urandom -Djava.awt.headless=true -Xmx1024m -XX:MaxPermSize=512m -XX:+UseConcMarkSweepGC"

PHP Java Bridge

http://php-java-bridge.sourceforge.net/

Install php-cgi:

sudo apt-get install php5-cgi libapache2-mod-php5

Test PHP Java bridge:

java -classpath JavaBridge.war TestInstallation
