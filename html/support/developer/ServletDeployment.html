Deployment to localhost for development

Right-click over the project and select Export -> war-file

create the war file where you have write-permission

Then copy the war file to the Tomcat folder

sudo cp JATServlet.war /var/lib/tomcat7/webapps

Make sure that the war file is readable by Tomcat

sudo chmod 666 /var/lib/tomcat7/webapps/JATServlet.war

Deployment to public server

install vsftp on server

sudo apt get install vsftp

follow instruction at

https://www.digitalocean.com/community/tutorials/how-to-set-up-vsftpd-on-ubuntu-12-04

on server, create directory named

/home/ftp

copy JATServlet to server

ftp 107.170.85.98

log in and issue command

send JATServlet.war

on server, copy file to

/var/lib/tomcat7/webapps

important: make sure the file has permissions -rw-rw-r-- and the owner is the same as the one where the web root is

chown webuser:webuser JATServlet.war

then restart tomcat

sudo service tomcat7 restart
